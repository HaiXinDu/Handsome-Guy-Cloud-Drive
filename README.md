# 局域网网盘

简单易用的局域网文件共享系统，支持文件和文件夹上传。

## 功能特点

- 📁 支持文件和文件夹上传
- 🕒 自动创建时间戳文件夹
- 🌐 局域网内多设备访问
- 📱 响应式界面
- 🚀 零配置启动

## 快速开始

### 方法一：使用启动脚本
双击 `启动服务器.bat` 即可自动启动服务

### 方法二：命令行启动
```bash
# 安装依赖
npm install

# 启动服务
npm start
```

## 使用方法

1. 启动服务器后，访问显示的地址
2. 拖拽文件到上传区域，或点击选择文件/文件夹
3. 文件将自动保存到 `uploads` 目录中

## 访问地址

- 本地访问：http://localhost:5000
- 局域网访问：http://[你的IP]:5000

## 文件存储

- 上传的文件保存在 `uploads` 目录
- 每次上传自动创建时间戳文件夹（格式：upload-2025-12-15T10-30-45）
- 保持原有的文件夹结构

## 技术栈

- Node.js + Express
- Multer（文件上传）
- 原生 HTML/CSS/JavaScript

## 系统要求

- Node.js 14+
- 现代浏览器（支持HTML5）
